Go Green With Us

Everybody these days are concerned about our environment and with good reason. You simply need only to look out of your window to see it happening... rain storms, drought, and all other natural environmental catastrophes appearing right where you live and work. That is why we at SSuite Office have designed and developed thee most energy efficient and least resource hungry software on the planet to help you save electrical energy and therefore reducing your carbon footprint as much as possible.


There is no one that can match our software bit for bit. Let me explain...

Our software has been designed and developed to run natively on any of Microsoft's Windows operating systems through the use of the WIN32 API structure. This makes our software do the least amount of work to satisfy your every whim and any work load that you might have.

Every function and feature of our software runs directly of the Windows code base; that means less cpu cycles, less hard drive access, and less memory usage. All of which consumes a certain amount of electrical energy from your power supply when being used. So where as with other so-called free and open source software, they all need to run on JAVA or .NET. This means that for every function, feature, or procedure, they first need to be translated by these third party operating systems before they can be run on your pc.

To be able to run open source software, JAVA or .NET first need to be installed and running on your operating system before they can be used, eating up more memory resources which needs to be refreshed every few nano seconds, thereby consuming more electrical energy. The more memory you use, the more energy your pc consumes to contain and run JAVA or .NET. For every feature or function you need to run, it first needs to be translated to a native Windows function using more cpu cycles and disk access in the process. This causes your system to consume even more electrical energy than what is required.

So you see... we are doing everything possible to make you consume less energy and making you more efficient in the process, even if other software companies and software developers are not.

Every little bit helps...

A Brief History on Our Free Software

Our humble website is a mere starting point for anyone who needs free software for their laptop, Android smartphone, Chromebook, or desktop PC. In today's retail market, you almost need to pay a king's ransom for software. It no longer needs to be this way, as open source developers and many other's have started to revolt against the major software development houses.

There are a million and one applications and software suites out there to be downloaded for free. I am just one of the many out there that can no longer afford to buy retail software, which is the main reason why I created my own software and office suites. Now I would like to share it with everybody else...

In some of my free office suites e.g. The Fifth Element / Excalibur, I have tried to implement as many usable software categories as possible (personal productivity, graphics, security, communication etc.). There are over 30 very useful applications to be found in my latest releases, more than any other office suite can provide, albeit not as functional as the more professional suites or software (that's why you pay for them).

Many people have discovered that you don't need all those advanced functions and features to write a simple letter, send an e-mail, create or edit a digital image, to use these professional software. A free application downloaded from the internet can do the same job without the cost. My software was designed to be as useful as possible to everyone, from the novice up to the most experienced user{and not to battle it out with the competition or try to replace them}. All this functionality and features are available in our free office software.

Please enjoy them all responsibly.
