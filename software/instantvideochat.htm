How to use IM Video Chat on any LAN, WAN, and Wi-Fi network

When you run this application, it will immediatly start your webcam if it is configured correctly.

If you receive an error on startup, please make sure your webcam's display codec is set to RGB, I420, or YUV2. Don't use any mpeg codecs as we don't support them.

To start, do the following:

  • Make sure to fill in all the information necessary in the settings window before starting... e.g. Ringtone, Message ID etc.
  • You simply either enter the I.P. address or the Host Name of the computer you wish to call.
  • If you don't know either one, just click on the ellipses "..." button above the first available window on top, and then select the "Who's online?" button.
  • A list of computer names will now be shown that is connected to your LAN.
  • When the list is complete, just click on the "Add Host Names to List" to add everyone to your address list.
  • Now simply double-click on the name you want to call, and then press the "Chat" button to start the connection. Make sure both computers are running the application.
  • The coloured window will now flash on the computer you are now calling, just press on the "Chat" button to start the messenger and Video connection.
  • Make sure to choose a ring tone for easier contact alerts. This can be done on the "Settings" button.

 

Completely compatible with all webcams, LANs, and Wi-Fi networks connected to your computer!
