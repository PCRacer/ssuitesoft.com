The best online/offline text editors on the internet!

Description...

This is the smallest text editor in the whole wide world, only 46KB in size. This text editor is full featured and is able to open, edit, and save *.html documents without a problem...

This makes our editor a very green option to use! The html document format is absolutely cross-platform compatible and only needs a web browser to open. This will bring the end to proprietary document formats and special and expensive software and applications necessary to open, edit, and save all your precious letters, memos, and all other types of documents.

Creating html documents will make sure they will always open and print exactly the way you intended them to be. Scaramouche Touch can either be downloaded and used offline, or simply click here to use it online!

With its online and offline capabilities, you don't need to install any software. Use the built-in capabilities of your HTML5 web browser for such things as spell checking, print, print preview, and find functions.

No more creating accounts, pay annual service fees, or social media account linking. Just download the file, double click it, and it will open in your default web browser, ready to be used in an instant!


Remember to set the downloads option in your web browser to always ask you where to save your documents. If not, then your documents will be saved to your default downloads folder.


Use the following web browsers for best user experience and performance:

  • Opera
  • Chromium
  • Google Chrome
  • Mozzila FireFox {Recommended}

If you experience any difficulty or problems with the editor, just update your web browser to the latest version!

Successfully tested on FireFox 45, Opera 36, and AppleWebKit/537.36 (KHTML, like Gecko), Chrome/49.0.2623.75, and Safari/537.36 OPR/36.0.2130.29.



Special features of Scaramouche Touch:

  • Insert tables of any size.
  • Undo and Redo actions.
  • Print and print preview your document.
  • Open, edit, and save any html document.
  • May be used to create web pages for websites.
  • Resize page area by dragging bottom right corner.
  • Insert images either by link, or just copy and paste.
  • Create and insert links for dynamic online documents.
  • View web page or html code with the click of a button.
  • Designed to be compatible with touch screen devices.
  • Open multiple instants of Scaramouche Touch in tab format.
  • Drag and drop content directly from open web pages e.g. text, images, or video etc.


Minimum System Requirements:

  • 1024 x 768 Display
  • HTML5 Compliant Web Browser
  • OS-X, iOS, Linux, Android, and Windows   -  32bit and 64bit


Known issues with HTML5 web browsers:

  • In FireFox, after saving a document, you need to start a new document and  reload the saved document to be able to continue working properly again.

Most HTML5 compliant browsers work very well, but not everyone implements functions and features the same way. So if you find a quirk or something strange, just either update to the latest version of your web browser, or let your web browser creators know about the problem. They will then fix it...

We have simply implemented HTML5 API to make our text editor work properly and as functional best we can.
